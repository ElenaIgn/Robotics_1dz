# :sparkles: Рассчет временной и пространственной сложности алгоритма (учебное проект)
____
## :pencil: Задание
Дано тестовое изображение (circles.png). Нужно на нем посчитать количество кругов  цветам (255, 0, 255 - BGR) с помощью двух разных алгоритмов (нужно самостоятельно придумать алгоритмы, а не использовать готовые из библиотеки). Для успешного выполнения задания нужно добиться того, чтоб алгоритмы работали корректно, сравнить их работу по трем видам сложности и написать отчет о полученных результатах.
